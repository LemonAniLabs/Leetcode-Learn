812. Largest Triangle Area

枚舉任意三個不同點，使用 shoelace formula 算三角形面積，持續更新最大值。這題點數小，O(n³) 的 brute force 就是標準解。

枚舉三點

橘色是目前候選三角形，綠色是目前找到的最大三角形。拖曳點會重新計算所有組合。

組合進度 1 / 10
目前面積 0
最大面積 0
最佳三點 -

核心公式

area(A,B,C) = |x1(y2-y3) + x2(y3-y1) + x3(y1-y2)| / 2

Time / Space Complexity
Time Complexity O(n³)
Space Complexity O(1)

演算法枚舉所有三點組合 `(i, j, k)`。組合數是 C(n, 3)，量級是 O(n³)。

每組三點的 shoelace 面積計算只做固定次數的乘法、加法與絕對值，所以每次是 O(1)。總時間因此是 O(n³)。

除了答案 `ans` 和迴圈索引之外，不需要額外資料結構，所以額外空間是 O(1)。

變數狀態

i, j, k 0, 1, 2
candidate P0, P1, P2
shoelace raw 0
max update false

程式碼對照

ans = 0
for i in range(n):
  for j in range(i + 1, n):
    for k in range(j + 1, n):
      area = triangleArea(points[i], points[j], points[k])
      ans = max(ans, area)
return ans
