643. Maximum Average Subarray I

固定長度 k 的 subarray 要最大 average，等價於找最大 window sum。每次滑動只加右邊新元素、移除左邊舊元素。

Fixed-size Sliding Window

藍色是目前長度 k 的 window，綠色是新加入元素，橘色是滑動時移除的元素。

步驟 0 / 0
window sum 0
max sum 0
answer 0

核心不變量

window_sum = sum(nums[left..right])
right - left + 1 = k
max_average = max_sum / k

容易漏掉：不要把 `max_sum` 初始化成 0，因為 nums 可能全部是負數。應該先用 `sum(nums[:k])` 當第一個 window。
Time / Space Complexity
Time Complexity O(n)
Space Complexity O(1)

每個元素最多被加入 window 一次、移除一次，所以時間是 O(n)。

只需要 `window_sum`、`max_sum` 和迴圈索引，不需要額外資料結構，所以空間是 O(1)。

狀態

window []
operation init

程式碼對照

window_sum = sum(nums[:k])
max_sum = window_sum
for i in range(k, len(nums)):
    window_sum += nums[i]
    window_sum -= nums[i - k]
    max_sum = max(max_sum, window_sum)
return max_sum / k
