Binary Tree Builder

貼上 LeetCode level-order array，包含 `null`，就能產生二元樹圖。適合搭配 traversal、path sum、LCA、BST 等題目快速看懂結構。

Level-order input

格式可用 `[5,4,8,11,null,13,4,7,2,null,null,5,1]`，也可省略中括號。`null`、`None`、`nil` 都會被視為空節點。

LeetCode array 如何變成樹

對節點 queue 做 BFS：
每取出一個非空 parent，就依序讀下一個值當 left，再下一個值當 right。

這和 LeetCode 題目描述中的 binary tree serialization 一致。`null` 代表該位置沒有節點，因此不會繼續產生它的 children。

Time / Space Complexity
Time Complexity O(n)
Space Complexity O(n)

解析 level-order array 時，每個輸入 token 最多處理一次，所以時間是 O(n)。

需要建立節點資料、queue 與座標資訊，最壞情況和節點數成正比，所以空間是 O(n)。
